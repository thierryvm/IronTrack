Pourquoi ce style

On ne rivalise pas avec Nike en copiant Nike. On rivalise avec Nike en ayant une voix.

L'univers fitness est saturé de dashboards Tailwind-indigo identiques. IronTrack v2 choisit l'inverse : une typographie éditoriale (Fraunces) pour les moments humains, une grotesque (Manrope) pour la précision, et un mono (JetBrains) pour les data. L'orange reste la signature. Le grain, le serif italique et les lignes de piste sont la personnalité. Zéro violet. Zéro AI-generic.
